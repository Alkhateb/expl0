#explo

###Bitcoin blockchain explorer
* Uses blockchain.info API
* Tracks live transactions
* Tracks live blocks and writes them to DB
    * when app is open (-.-)
* You can check:
    + transaction info
    + block info
    + address balance
   
   
+ Made for learning purposes
    + implementing web sockets
    + working with REST api
    + slim framework
    + sqlite
